    1. Introduction to HTML

      2. HTML was created by tim berners-Lee in 1999 and latest version is HTML 5. It is the most basic language and simple to learn and modify. HTML also defines the structure of web pages, means we can create a website using html which can be viewed on internet-connected devices like laptops, android mobile phones, etc.

    2. Definition of html

      3. HTML stands for: Hypertext Markup Languge. It basically refers to a text that is being stored in a computer system that allows the user to move from one piece of text or document to another as figure out below.

      But here let's breakdown the term "HTML" to see what exactly it refers to!
      HTML means Hypertext Markup Languge as said above, but what does that means?
      • Hypertext
        • refers to the text that is not restricted to a sequential format and that includes links to other text. The links can connect online pages inside a single or different websites.
      • Markup Language
        • is a language that is interpreted by the browser and it defines the elements within a document using "tags". It is also human-readable, which means that markup files use common words rather than the complicated syntax of programming languges.
